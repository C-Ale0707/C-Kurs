Detailed Description

Headerfile mit Datenstruktur und Zugriffsfunktionen fuer FIR-Filter.

In diesem Header wird die Datenstruktur struct fir_filter_variablen definiert, welche interne Groessen des FIR-Filters aufnimmt. Ausserdem werden Funktionsprototypen bereitgestellt fuer den Konstruktor, den Destruktor, eine Funktion zur Eingabe der Filterkoeffizienten sowie eine Funktion zur Berechnung des Filters.
